CSS Font Attributes

How basic CSS attributes can change the look of fonts.

Attribute Effect on Font Code Notes
Font Family Determines the Typeface font-family: example; Options Here
Font Size Determines the size of text font-size: ##px; px=pixels
Font Style Can add italics font-style: italic; Options: normal, italic, oblique
Font Weight Can bold font font-weight: bold; Options: normal, bold
Font Variant Can show font in small caps font-variant: small-caps; Options: normal, small-caps
The following is a way to combine font attributes in a concise way.
Font Shorthand Combines any attribute listed above font: style variant weight size/line-height, family; Must be typed in order shown. Must include size and family.For Reference

Other attributes that impact the characteristics of the text. These use the word Text in the Code.

Attribute Effect on Text Code Notes
Text Color Determines the Color of the Text color: example; Can use Color Name, RGB, and Hexadecimal code.Details Here
Text Alignment Determines the Position of the Text text-align: center; Options: Right, Center, Left or Justified
Text Decoration Places or Removes lines from the Text text-decoration: line-through; Options:overline, line-through, underline, or none. None will remove underline from a link.
Text Transformation Determines the Capitalization of Text text-transformation: uppercase; Options: UPPERCASE, lowercase, Capitalize
